# Matrix Multiplication & Determinant Calculation

这是一个教学/演示型仓库，包含若干用于矩阵乘法与行列式计算的交互式 Python 脚本。所有脚本均为交互式运行（从命令行通过 input 读取数据），仓库未采用命令行参数或文件输入（请以交互式提示为准）。

## 文件清单与作用

- `matrix_mul.py`：交互式矩阵乘法。提示用户输入 A 与 B 的行数/列数，再逐项输入元素（当前实现读取整数）；使用行×列点积计算并格式化打印结果。
- `det_cal_1.py`：基于递归余子式（Laplace expansion）计算行列式。先输入阶数，然后按元素输入（整数）。适合非常小的矩阵用于教学演示。
- `det_cal_2.py`：基于高斯消元（Gaussian elimination）将矩阵化为上三角后计算行列式。先输入阶数，然后输入矩阵元素（当前实现接受数值）。时间复杂度约 O(n^3)。
- `det_cal_3.py`：动态逐行输入的行列式计算器（不需要事先输入阶数）。按行输入元素，行内用空格分隔；输入空行或 `end` 结束输入。支持浮点数，内部使用高斯消元法并在输出时保留 6 位小数。
- `app.py`：一个简单的交互式菜单，统一演示矩阵乘法（调用 `matrix_mul` 的交互流程）与行列式计算（调用 `det_cal_2` 的交互流程），适合作为课堂或手动演示入口。
- `Develop process.ipynb`：开发过程笔记（Jupyter Notebook），包含实验与草稿代码，可在 Notebook 中阅读。

## 运行环境

- Python 3.8+（已在 Windows PowerShell 下测试过交互流程）。
- 无外部依赖；所有脚本使用标准库和纯 Python 实现算法。

## 快速运行（示例，PowerShell）

在项目根目录打开 PowerShell，运行相应脚本：

矩阵乘法：

```powershell
python .\matrix_mul.py
```

递归行列式（小阶教学示例）：

```powershell
python .\det_cal_1.py
```

高斯消元行列式：

```powershell
python .\det_cal_2.py
```

动态逐行输入行列式（支持浮点）：

```powershell
python .\det_cal_3.py
```

统一演示入口（菜单）：

```powershell
python .\app.py
```

运行 `app.py` 后会出现一个简单菜单：输入 1 进入矩阵乘法交互流程，输入 2 进入行列式计算交互流程。

## 输入格式与交互示例

以下是每个脚本的交互输入说明与示例（均为人工交互输入）：

- matrix_mul.py
  - 先输入：A 的行数（整数）、A 的列数、B 的行数、B 的列数。
  - 程序会检查 col_A == row_B，若不满足则退出并提示错误。
  - 然后逐项输入 A[i][j] 与 B[i][j]（当前实现读取整数）。
  - 简要示例：

    界面：
    矩阵A行数：2
    矩阵A列数：3
    矩阵B行数：3
    矩阵B列数：2
    A[1][1]=1
    ...

- det_cal_1.py
  - 先输入阶数 n（整数），随后按行/按元素逐个输入矩阵元素（整数）。

- det_cal_2.py
  - 先输入阶数 n，再逐个输入矩阵元素（实现以数值类型处理，建议输入整数或浮点）。

- det_cal_3.py
  - 逐行输入矩阵，每行元素以空格分隔（支持浮点数）。
  - 输入空行或输入 `end` 表示结束输入。脚本会校验是否为方阵（行数 == 列数）。
  - 示例：

    请逐行输入行列式元素（每行元素用空格分隔，输入空行或'end'结束）：
    1 2 3
    4 5 6
    7 8 9

    输出：3阶行列式的值为：0.0

## 算法说明与复杂度

- det_cal_1.py：使用 Laplace 展开（按余子式递归），时间复杂度为指数级；适合教学与小阶矩阵（n 很小时）演示。
- det_cal_2.py / det_cal_3.py：使用高斯消元（上三角化）计算行列式，时间复杂度约为 O(n^3)，数值稳定性取决于主元选择（当前实现使用简单的最大绝对值行选择）。
- matrix_mul.py：使用朴素三重循环（行-列点积）计算矩阵乘法，时间复杂度为 O(rows_A * cols_A * cols_B)。
